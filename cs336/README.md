# CS336 Track

This directory is reserved for the next stage of the project: moving from a
book-based from-scratch implementation toward CS336-level language modeling
systems and research challenges.

Planned topics include tokenizer training, Transformer implementation details,
training performance, profiling, scaling behavior, data pipelines, evaluation,
and alignment-related experiments.

## Layout

- `notes/`: lecture notes and reading summaries written in my own words.
- `experiments/`: small reproductions, checks, and training-system experiments.
- `figures/`: plots and diagrams produced from my own experiments.
- `lectures/`: optional local clone of `stanford-cs336/lectures`; ignored by git.
