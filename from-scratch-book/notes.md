# From-Scratch Book Notes

This section records the current stage of implementing a GPT-style language
model from scratch.

## Current Artifact

- `scratch_again.ipynb`: the main notebook for the current milestone.
- `llm-from-scratch.ipynb`: earlier notebook work from the same book track.
- `ch02/` to `ch05/`: chapter-level implementation notes and scripts.
- `figures/`: reserved for self-produced experiment plots and visualizations.
- `artifacts/`: local checkpoints and generated model files. These are not
  intended for git tracking.

## Suggested Figure Ideas

- training or validation loss curves
- causal attention mask visualization
- temperature sampling comparison
- token embedding norm distribution
- context length effect on generation quality
- gradient norm tracking during training
